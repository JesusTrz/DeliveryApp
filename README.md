# AppDelivery

Aplicación de delivery desarrollada con React + Vite como proyecto académico para la materia de **Administración de Proyectos** en la UTTT.

## Propósito del proyecto

El objetivo principal de este proyecto es aprender y practicar la **colaboración en equipo** utilizando herramientas de gestión y control de versiones ampliamente usadas en la industria:

- **Zoho** — Gestión de proyectos, seguimiento de tareas y comunicación del equipo.
- **Jira** — Planificación de sprints, tableros ágiles y seguimiento de incidencias.
- **GitHub** — Control de versiones, revisión de código mediante Pull Requests y flujo de trabajo colaborativo.

## Tecnologías

- [React](https://react.dev/) con [Vite](https://vite.dev/) para el desarrollo del frontend.
- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) — usa [Oxc](https://oxc.rs) para transformación rápida.

## Inicio rápido

```bash
npm install
npm run dev
```
